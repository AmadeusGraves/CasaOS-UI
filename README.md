# CasaOS-UI

The front-end of CasaOs,build with VueJS

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
Will be output to the ../web folder

### Lints and fixes files
```
yarn lint
```
